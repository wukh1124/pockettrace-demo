# 登入測試帳號 Workflow

此文件記錄了開發與測試時使用的固定帳號資訊。

## 測試帳號資訊

- **電子郵件**: `test@example.com`
- **密碼**: `Password123!`

## 登入步驟

1. 啟動開發伺服器 (`npm run dev`)。
2. 前往 [http://localhost:3000/login](http://localhost:3000/login)。
3. 輸入上述電子郵件與密碼。
4. 點擊「登入」即可進入系統。

> [!NOTE]
> 如果此帳號尚未建立，請先前往註冊頁面使用相同資訊註冊一次。
